# `config`

A small shared library exposing design tokens and text styles used across apps (colors, font enums and reusable text mapping helpers).

- `index.ts` exports library modules for convenient imports (example :  `import { textStyles } from "@repo/config"`).
- `styles.ts` contains the enums, types and the `textStyles` mapping that define font families, sizes, weights, and other text related tokens.
